# Sistema-de-Alerta-de-Clima
O sistema foi criado para monitorar sensores de temperatura e umidade e avaliar automaticamente se os valores recebidos ultrapassam limites pré-definidos. A ideia simula uma arquitetura em nuvem semelhante à utilizada na AWS, com foco em processamento assíncrono, filas, auditoria e notificações de alerta.

O objetivo principal é permitir que sensores enviem leituras em tempo real e que o sistema consiga processar esses dados de forma eficiente, gerar alertas quando necessário e manter um registro completo para auditoria.
